<h1>Bluebot commands</h1>
This project is a custom alternative to the original StreamElements bot commands list for BlueAgua on Twitch. StreamElements can only display a list of its own enabled commands, and since there could be multiple active bots (this is the case), it was necessary a new option to fully display ALL the commands across all the bots.

<hr>
This webpage has been developed using Vite, tailwindcss, typescript
